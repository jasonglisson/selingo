== Description ==
Wordpress Facebook Conversion Pixel plugin allows users to embed Facebook conversion pixel script to wordpress website. You can track woocommerce, cart, checkout, product and any wordpress page/post.

**New: Option to get Conversion pixel tracking results**

To create your Facebook pixel:

<a href="http://www.miraclewebsoft.com/facebook-conversion-pixel-2/">Check Video Tutorial</a>

<ol>
<li>Go to your <a target="_blank" href="https://www.facebook.com/ads/manager/pixel/facebook_pixel"> Facebook Pixel tab</a> in Ads Manager.</li>
<li>Click Create a Pixel.</li>
<li>Enter a name for your pixel. You can have only one pixel per ad account, so choose a name that represents your business.</li>
<li>Note: You can change the name of the pixel later from the Facebook Pixel tab.</li>
<li>Check the box to accept the terms.</li>
<li>Click Create Pixel.</li>
<li>Copy code and paste</li> 
</ol>
